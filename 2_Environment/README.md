Contains the project roadmap with development phases and detailed use cases for different user scenarios.

## Use Cases

### Use Case 1: Content Creator
**Goal**: Quickly prepare post-production assets for a YouTube video
**Steps**: 
1. Open postproduction_helper.html
2. Load project files from 2026-02-14/input/
3. Review and edit chapter markers
4. Generate description with timestamps
5. Export subtitle file

### Use Case 2: Video Editor
**Goal**: Organize B-roll and graphics for editing
**Steps**:
1. Review shotlist and storyboard
2. Match graphics to timeline markers
3. Export EDL for editing software
4. Verify all assets are present

### Use Case 3: Producer
**Goal**: Quality check all deliverables
**Steps**:
1. Review transcripts and subtitles
2. Verify chapter markers align with content
3. Check music cues and licensing
4. Approve final asset package
